Stars Map

About the App

Stars Map is an educational application designed to help users explore the solar system in an interactive and visual way. The app simulates planetary motion using simplified scientific models to provide a realistic and engaging learning experience.

Features

Realistic Planet Motion

Interactive Exploration

Educational Content
